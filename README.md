# Имитационное моделирование ТГУ
---

## 1. Структура курса

Курс поделен на **три темы (четверти)** в соответствии с содержанием:

- **Тема 1:** лабораторные работы №1–3  
- **Тема 2:** лабораторные работы №4–7  
- **Тема 3:** лабораторные работы №8–10  

Каждая лабораторная работа оценивается по **5-балльной шкале**.

> **Важно:**  
> Во время сдачи лабораторной работы студенту задаются вопросы по теме работы.  
> **Наличие рабочего кода НЕ является достаточным условием для зачёта.**

---

## 2. Оценивание по темам

По завершении каждой темы студент получает **среднюю арифметическую оценку** лабораторных работ, входящих в данную тему.

---

## 3. Сроки сдачи лабораторных работ

- Сдача лабораторных работ происходит **в установленные преподавателем сроки**.
- **Все лабораторные работы обязательны к сдаче**.
- В случае сдачи лабораторной работы **не в срок без уважительной причины**, студенту дополнительно задаётся вопрос по соответствующей теме.

---

## 4. Экзамен и автомат

### Экзамен

Экзаменационный билет состоит из **трёх вопросов**, соответствующих трём темам курса:

- вопрос по Теме 1  
- вопрос по Теме 2  
- вопрос по Теме 3  

Каждый вопрос оценивается по **5-балльной шкале**.

**Итоговая оценка за экзамен** — среднее арифметическое оценок по всем вопросам билета.

---

### Автомат

Если в конце семестра:

- **все лабораторные работы сданы в срок**, и
- **средняя оценка по каждой теме ≥ 2.5**,

то студент получает **соответствующие оценки по темам экзаменационного билета автоматически**.

> Если студент желает повысить оценку, он может идти на экзамен.

---

### Обязательный экзамен

Если в конце семестра:

- по любой теме средняя оценка **< 2.5**, **или**
- хотя бы одна лабораторная работа сдана **не в срок без уважительной причины**,

то студент **обязан идти на экзамен**.

При этом:
- темы, по которым средняя оценка **≥ 2.5** и оценка студента устраивает, **можно не сдавать**.
